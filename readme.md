# Remote Test App

## Цель приложения:
* Предоставить удобное приложение для удаленного редактирования и прохождения теста
* Сделать обучение более интересным и удобным

## Используемые технологии:
* VueJS
* NodeJS
* HTML
* CSS
* Javascript

## Установка на сервер
Дополнительную информацию можно найти по ссылке [Развертывание на продакшн](https://ru.vuejs.org/v2/guide/deployment.html)

## Необходимые шаги для установки на сервер
* Скопировать папку себе локально на ПК
* Установить и развернуть программу XAMPP
* Установить папку сайта в public часть программы XAMPP
